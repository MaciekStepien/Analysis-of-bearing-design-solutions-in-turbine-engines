## About the Project

This Bachelor's pre-thesis project aims to analyze and compare different bearing design solutions applied in turbine engines, covering their mechanical properties, advantages, and limitations.

## Repository Structure

- `README.md` – Project overview and instructions  
- `Analiza rozwiązań konstrukcyjnych łożyskowania w silnikach turbinowych.pdf` – Original thesis in Polish

You can browse the thesis directly in this repository or download PDF versions for offline reading.
